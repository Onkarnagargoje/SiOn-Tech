# SiOn Tech Solutions

Premium software agency website — React, Vite, Tailwind CSS, GSAP, Three.js / React Three Fiber, Lenis, and Framer Motion.

## Quick start

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |

## Stack

- React 19 + Vite 8
- Tailwind CSS 3
- GSAP + ScrollTrigger
- Three.js / R3F / Drei
- Lenis smooth scroll
- React Router + React Helmet Async
- Lucide icons

## Structure

```
src/
  components/   # layout, ui, animations, three, home
  data/         # content & site config
  hooks/        # Lenis, media, mouse, count-up
  pages/        # all routes
```

## Pages

Home, About, Services, Portfolio (+ project detail), Case Studies, Technologies, Careers, Blog (+ post), Contact, Privacy, Terms, 404.
